###  [Blog 1 - Building Your First Search Application with Amazon OpenSearch Service](3.1-Blog1/)
This blog provides a practical introduction to Amazon OpenSearch Service and demonstrates how to build a secure, scalable search application on AWS. It covers the fundamental concepts of OpenSearch, including documents, indices, shards, and search ranking, while also walking through a production-ready architecture using AWS App Runner, Amazon Cognito, API Gateway, Lambda, and OpenSearch. Whether you're developing search-driven applications, analytics platforms, or real-time data solutions, this guide offers a solid starting point for leveraging OpenSearch in the AWS ecosystem.


###  [Blog 2 - Automate medical record digitization with Amazon Bedrock Data Automation and AWS HealthLake](3.2-Blog2/)
This blog explores how to automate the digitization of medical records using Amazon Bedrock Data Automation and AWS HealthLake. ###  [Blog 3 - How AI and Robotics Are Transforming Sustainable Agriculture with Amazon SageMaker AI](3.3-Blog3/)
This blog delves into the journey of Aigen, an agricultural robotics company, and how they transformed their machine learning pipeline using Amazon SageMaker AI.
