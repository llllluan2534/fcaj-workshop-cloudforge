### Mục tiêu tuần 3:

* Hiểu tổng quan về Amazon EC2, các thành phần cơ bản và các loại instance type.
* Nắm vững cách kết nối vào EC2 instance và cấu hình Security Group.
* Áp dụng user data để chạy script bootstrap khi instance khởi động.
* Khám phá cách sử dụng lưu trữ EBS và quản lý chi phí tài nguyên EC2.
* Thực hành khởi tạo, kiểm tra, kiểm thử và dọn dẹp EC2 instance.

### Kết quả đạt được tuần 3:

* Nắm bắt được nền tảng vững chắc về Amazon EC2 và các thành phần của nó.
* Thực hành thành công việc tạo, cấu hình và kết nối vào EC2 instance.
* Thiết lập an toàn với Security Group và kiểm soát truy cập hiệu quả.
* Ứng dụng thành công User Data để tự động hóa cấu hình instance bằng bootstrap script.
* Làm quen với việc thao tác trên EBS volume và quản lý dữ liệu.
* Nắm vững nguyên tắc quản lý chi phí bằng cách thực hiện dọn dẹp tài nguyên triệt để sau các bài thực hành.
