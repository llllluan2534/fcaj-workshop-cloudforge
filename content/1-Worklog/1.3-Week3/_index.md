### Week 3 Objectives:

* Understand the overview of Amazon EC2, basic components, and instance types.
* Master connecting to EC2 instances and configuring Security Groups.
* Apply user data to run bootstrap scripts when the instance boots up.
* Explore how to use EBS storage and manage EC2 resource costs.
* Practice launching, verifying, testing, and cleaning up EC2 instances.
