**Week 1:** [Onboarding, AWS Fundamentals & IAM Basics](1.1-week1/) - Getting started with AWS, setting up accounts, and understanding core Identity and Access Management concepts.

**Week 2:** [Amazon VPC, Site-to-Site VPN & IaC](1.2-week2/) - Building isolated virtual networks, establishing secure VPN connections to on-premises, and an introduction to Infrastructure as Code.

**Week 3:** [Amazon EC2, Security Group & EBS](1.3-week3/) - Deploying virtual servers, managing network access controls, and attaching block storage volumes.

**Week 4:** [Amazon RDS](1.4-week4/) - Understanding managed relational databases and deploying a highly available database instance.

**Week 5:** [EC2 Auto Scaling Group & Load Balancer](1.5-week5/) - Ensuring application high availability and scaling dynamically based on traffic demand.

**Week 6:** [Amazon CloudWatch & IAM Identity Center](1.6-week6/) - Monitoring resources, setting up alarms, and centralizing workforce access management.

**Week 7:** [Route 53 Resolver & Hybrid DNS](1.7-week7/) - Managing DNS resolution and setting up hybrid routing between AWS and on-premises networks.

**Week 8:** [AWS CLI](1.8-week8/) - Mastering the command line interface to automate and manage AWS resources efficiently.

**Week 9:** [Amazon Lightsail](1.9-week9/) - Deploying simple web applications and virtual private servers rapidly using Lightsail blueprints.

**Week 10:** [IAM Role & AWS Cloud9](1.10-week10/) - Applying the principle of least privilege using IAM roles and setting up a cloud-based IDE.

**Week 11:** [Amazon S3](1.11-week11/) - Working with scalable object storage, managing buckets, and implementing data lifecycle policies.

**Week 12:** [First Cloud AI Journey Overview](1.12-week12/) - A comprehensive review of the entire journey, connecting the dots between all learned services.
